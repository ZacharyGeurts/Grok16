Architecture

Grok16 @ 16.1.0 introduces a unified g16 driver — one binary for C and C++. GCC backends live under libexec/grok16/.

Unified driver (16.1.0)

bin/g16          → driver/g16-unified.c (exec dispatch)
bin/g++16        → symlink → g16
libexec/grok16/g16-cc   ← relocated GCC C driver
libexec/grok16/g16-cxx  ← relocated GCC C++ driver

Detection: .c / -std=gnu17 → g16-cc; .cpp / g++16 / -std=gnu++26 → g16-cxx. Self-host rebuild uses backends directly as CC/CXX.
