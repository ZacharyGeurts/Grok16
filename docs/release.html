Release 5.0.1 — version one

Grok16 5.0 kicks off as v1.0 operator framing. Compiler g16 @ 16.2.0. Tag v5.0.1. Prior 2.x–4.x remain in git history only.

Field platform: Do not create field files — use the 2D auto-field plane.

What ships

  • belt_2_0 single fabric — 8192 redata chunk, 512 die slots
  • field_physics — thermal guard, no -ffast-math for production
  • Binary package — g16 + AmmoCode + signed settings (binary-package)
  • 2D field platform — data/grok16-field-platform-doctrine.json
  • ZNetwork field wire — egress convert / ingress deconvert design
  • Speed bench v5.0.0 — best exec 102.84M ops/s
  • Compiler symlinks · build-essential fabric · AmmoCode field instill
  • 5.0.1 — AmmoOS — ammoos profile, integrate-ammoos, verify-ammoos-surfaces (pairs AmmoOS 1.9.9e)

Checkout

git clone https://github.com/ZacharyGeurts/Grok16.git
cd Grok16 && git checkout v5.0.1
export G16_PREFIX="$(pwd)" G16_BELT_PROFILE=belt_2_0
./scripts/grok16-toolchain.sh rebuild
./scripts/grok16-toolchain.sh verify
./scripts/grok16-toolchain.sh test-battery-belt
./scripts/grok16-toolchain.sh bench-refresh

Binary package

./scripts/grok16-toolchain.sh binary-package
tar xzf dist/grok16-5.0.1-linux-x86_64.tar.gz
cd grok16-5.0.1-linux-x86_64 && source grok16-env.sh
./share/ammocode/ammocode

Benchmarks

./scripts/grok16-toolchain.sh bench-triad
./scripts/grok16-toolchain.sh exec-full-bench
