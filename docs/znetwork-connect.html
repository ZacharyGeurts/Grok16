Field expansion down the wire

[Field A] → field-io packet (sealed) → wire → verify + deconvert → [Field B]

Never ship raw field files on the wire. Modules: field-io-packet.py, packet-field.py, connection-gatekeeper.py.

Identity without cellphone

  • Display name — public label only, not root of trust
  • Wire-point — opaque HMAC token, rotates with sovereign receipt
  • LAN beacon — polite discovery (AmmoCode port 9555)
  • Invite URL — one-time collab token (WS :9556)
  • USB sovereign key — optional physical anchor (no SMS)
