CMake usage

cmake -S myproject -B build \
  -DCMAKE_TOOLCHAIN_FILE=/path/to/Grok16/cmake/grok16-toolchain.cmake \
  -DCMAKE_PROJECT_INCLUDE=/path/to/Grok16/cmake/grok16-profile-field-opt.cmake

Bench profile selection

G16_BENCH_PROFILE=field_opt ./scripts/grok16-toolchain.sh bench
G16_FIELD_SPEED=1 ./scripts/grok16-toolchain.sh field-bench

Flag emission

python3 scripts/grok16-profile-flags.py field_opt cxx
python3 scripts/grok16-profile-flags.py field_opt link
python3 scripts/grok16-profile-flags.py field_opt source

PROFILE link — LTO/link flags (normalized).

PROFILE source — bench .cpp path from profiles.json.

PROFILE defs — -D macros only.

Safety mandate

All field targets should include cmake/g16-field-mandate.cmake (fortify, stack protector, RELRO, PIE). Used by World_Redata L2.
