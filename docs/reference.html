Reference

Grok16 @ 16.1.1 — unified g16 driver. See Master Coder C and Master Coder C++ for full command indexes.

Unified g16 driver

PathRole
bin/g16Single front door — auto-detects C vs C++ from argv[0], -std=, -x, extensions, -lstdc++
bin/g++16Symlink to g16 (compat alias; forces C++ detection via basename)
libexec/grok16/g16-ccRelocated GCC C backend (real ELF)
libexec/grok16/g16-cxxRelocated GCC C++ backend (real ELF)

Override: G16_FORCE_C=1 or G16_FORCE_CXX=1. Defaults: G16_C_STD=gnu17, G16_CXX_STD=gnu++26.
